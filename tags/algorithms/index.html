Fibbonacci Numbers

2022-07-06
#algorithms  #code 
link to problem Problem explanation The problem is asking us to return the respective fibbonacci number of the input. Where the output should be given by the function f(0) = 0, f(1) = 1, f(n) = f(n-1) + f(n-2). Solution This is a common example of a problem that is best solved using recursion. We start by defining our base cases. if n == 0: return 0 elif n == 1: return 1 These are our stop conditions.
